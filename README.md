# Реплей «Отладка»

В этом задании тебе предстоит освоить отладку программ на C# в Visual Studio, а затем продемонстрировать этот навык преподавателю на занятии.

Для начала скачай [исходный код программы](https://github.com/kontur-courses/debug-replay/archive/master.zip).

Используй [пошаговые инструкции](https://docs.google.com/document/d/1OpGeDvzUClXODYu6W0RMhDS08sOXAN772Wu-dak4lOs/edit?usp=sharing), чтобы научиться выполнять отладку программы так, как в этом [видео](https://youtu.be/G5NYJUD2cj4).
Открой код программы в Visual Studio и выполни все инструкции. Параллельно можешь запустить видео с правильным вариантом, чтобы подсматривать в случае затруднений или просто для того, чтобы сделать все правильно.

Выполни отладку по пошаговым инструкциям несколько раз, чтобы закрепить навык.

Чтобы сдать задание преподавателю, останется научиться выполнять упрощенную версию задания по [инструкциям показа преподавателю](https://docs.google.com/document/d/1fr2F_TTwdTeSuUYsjKGfVno9ZocY0q337q-q-V2vJEQ/edit?usp=sharing).
В этих инструкциях написано, что конкретно надо будет показать преподавателю, а что ты выполнишь сам, пока преподаватель будет делать что-то еще. Например, принимать этот же реплей, но у другого студента.
При сдаче можно будет открыть инструкции на любом устройстве.

В результате прохождения реплея ты освоишь и закрепишь несколькими повторениями навык отладки.
